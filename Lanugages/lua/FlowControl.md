

## if-else

```lua
if condition then
  print("yes")
elseif condition then
  print("maybe")
else
  print("no")
end
```

## Loops

```lua
-- notice do .. end
while condition do
end

for i = 1,5 do
end

for i = start,finish,delta do
end

for k,v in pairs(tab) do
end

repeat
until condition

-- Breaking out:
while x do
  if condition then break end
end
```
